/tmp demo engine
================

Making demos for fun and profit.

How to run
----------

Make sure you have the following installed and available from PATH:
 * sh
 * python
 * ruby
 * java 7 (from Oracle)
 * node/npm

Then run the following commands:
```sh
npm install
node .
```

You can now open a browser at <http://localhost:8084> to use the editor.
